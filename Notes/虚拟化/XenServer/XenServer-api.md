# 主机

```
xe host-reboot     #重启主机
xe host-shutdown   #关闭主机
xe host-disable         #禁用指定的XenServer
```

# VM

```
xe vm-reboot       #重启VM
xe vm-recover      #从vdi中包含的数据库恢复VM
xe vm-unpause      #恢复指定的VM
xe vm-shutdown     #关闭指定的VM
xe vm-start        #启动指定的VM
xe vm-pause        #暂停指定的VM
```
